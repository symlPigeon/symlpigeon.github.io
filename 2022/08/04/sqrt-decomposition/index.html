# 数论分块

对于包含了向下取整的除法式，例如∑i=1n⌊nf(x)⌋\sum_{i=1}^{n}\lfloor \cfrac{n}{f(x)}\rfloor∑i=1n​⌊f(x)n​⌋ 这样的算式，直接从111 到nnn 去枚举计算显然效率是比较低的。而数论分块也就给予了可以快速计算的方法。

我们可以注意到向下取整式有这样的性质：

∀a,b,c∈Z,⌊abc⌋=⌊⌊ab⌋c⌋\forall a,b,c\in\mathbb{Z},\lfloor\frac{a}{bc}\rfloor=\lfloor\frac{\lfloor\frac{a}{b}\rfloor}{c}\rfloor∀a,b,c∈Z,⌊bca​⌋=⌊c⌊ba​⌋​⌋

证明

向下取整实际上可以写成这样的形式：

a=⌊a⌋+r (0≤r<1)a=\lfloor a\rfloor+r\ (0\le r<1)a=⌊a⌋+r (0≤r<1)

因此上面式子带入之后可以得到：

⌊abc⌋=⌊1c⋅ab⌋=⌊1c⋅(⌊ab⌋+r)⌋=⌊1c⌊ab⌋+rc⌋=⌊1c⌊ab⌋⌋□\begin{aligned} \lfloor\frac{a}{bc}\rfloor&=\lfloor{\frac{1}{c}\cdot\frac{a}{b}}\rfloor&\\&=\lfloor{\frac{1}{c}\cdot(\lfloor\frac{a}{b}\rfloor+r)}\rfloor&\\ &=\lfloor \frac{1}{c}\lfloor\frac{a}{b}\rfloor+\frac{r}{c}\rfloor&\\&=\lfloor \frac{1}{c}\lfloor\frac{a}{b}\rfloor\rfloor&\\ &&\square \end{aligned}⌊bca​⌋​=⌊c1​⋅ba​⌋=⌊c1​⋅(⌊ba​⌋+r)⌋=⌊c1​⌊ba​⌋+cr​⌋=⌊c1​⌊ba​⌋⌋​□​

通过上面的性质，我们可以得到下面的结论：

定理（数论分块）: 对于常数nnn 和一个i≤ni\le ni≤n，使得

⌊ni⌋=⌊nj⌋\lfloor\frac{n}{i}\rfloor=\lfloor\frac{n}{j}\rfloor⌊in​⌋=⌊jn​⌋

成立的最大的j(i≤j≤n)j(i\le j\le n)j(i≤j≤n) 为⌊n⌊ni⌋⌋\lfloor\cfrac{n}{\lfloor\frac{n}{i}\rfloor}\rfloor⌊⌊in​⌋n​⌋。

证明过程说实话有些偏。

证明

设k=⌊ni⌋k=\lfloor\cfrac{n}{i}\rfloork=⌊in​⌋，有k≤⌊ni⌋k\le\lfloor\cfrac{n}{i}\rfloork≤⌊in​⌋。

而⌊nk⌋≥⌊n⌊ni⌋⌋=⌊i⌋=i\lfloor\cfrac{n}{k}\rfloor\ge\lfloor\cfrac{n}{\lfloor\frac{n}{i}\rfloor}\rfloor=\lfloor{i}\rfloor=i⌊kn​⌋≥⌊⌊in​⌋n​⌋=⌊i⌋=i，故有j=imax=⌊n⌊ni⌋⌋.j=i_{max}=\lfloor\cfrac{n}{\lfloor\frac{n}{i}\rfloor}\rfloor.j=imax​=⌊⌊in​⌋n​⌋.

这个定理说明了这样的一个问题，例如我们要计算∑i=1nf(x)⌊ni⌋\sum_{i=1}^nf(x)\lfloor\cfrac{n}{i}\rfloor∑i=1n​f(x)⌊in​⌋ 的时候，如果计算f(x)f(x)f(x) 前缀和即s(i)=∑j=1if(j)s(i)=\sum_{j=1}^if(j)s(i)=∑j=1i​f(j) 的时间复杂度并不高，那么我们可以快速的计算这个和式：对于iii，我们找到满足⌊ni⌋=⌊nj⌋\lfloor\cfrac{n}{i}\rfloor=\lfloor\cfrac{n}{j}\rfloor⌊in​⌋=⌊jn​⌋ 的最大的j=⌊n⌊ni⌋⌋j=\lfloor\cfrac{n}{\lfloor\frac{n}{i}\rfloor}\rfloorj=⌊⌊in​⌋n​⌋，那么∑k=inf(x)⌊nk⌋=⌊ni⌋⋅(s(j)−s(i))\sum_{k=i}^nf(x)\lfloor\cfrac{n}{k}\rfloor=\lfloor\cfrac{n}{i}\rfloor\cdot(s(j)-s(i))∑k=in​f(x)⌊kn​⌋=⌊in​⌋⋅(s(j)−s(i))。

以这个题目为例：

给出正整数nnn 和kkk，计算：

G(n,k)=∑i=1nkmod  iG(n,k)=\sum_{i=1}^n{k\mod i}G(n,k)=i=1∑n​kmodi

数据范围为10910^9109，在 OI 的时限内使用O(n)O(n)O(n) 的算法必然是不现实的。

我们考虑到amod  b=a−⌊ab⌋⋅ba\mod b=a-\lfloor\cfrac{a}{b}\rfloor\cdot bamodb=a−⌊ba​⌋⋅b，那么上式可以转换为：

G(n,k)=∑i=1nk−⌊ki⌋⋅iG(n,k)=\sum_{i=1}^n{k-\lfloor\cfrac{k}{i}\rfloor\cdot i}G(n,k)=i=1∑n​k−⌊ik​⌋⋅i

这样子式子就被转换成了数论分块能够解决的形式。

显然∑i=1nk\sum_{i=1}^nk∑i=1n​k 是容易计算的，同时函数f(i)=if(i)=if(i)=i 的前缀和也非常容易计算，通过数论分块我们能够快速解决这个问题。

不过这道题这边有些不一样，求和的上界是nnn 而处于向下取整分子上的是kkk。我们可以加入判断，如果k>nk>nk>n，那么枚举到i>ni>ni>n 的时候自然停止就可以，而如果n>kn>kn>k，那么在i>ki>ki>k 的时候自然停止就可以，因为后面的⌊ki⌋⋅i=0\lfloor\cfrac{k}{i}\rfloor\cdot i=0⌊ik​⌋⋅i=0，没有必要计算。故我们循环终止的条件是i>min⁡(n,k)i>\min(n,k)i>min(n,k)。

这样，这道题就可以解决了！
