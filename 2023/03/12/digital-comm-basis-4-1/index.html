# AWGN 信道

一个最基本的信道模型应该就是加性高斯白噪声（Additive White Gaussian Noise）信道，对这个信道的表述是：

r(t)=sm(t)+n(t)r(t) = s_m(t)+n(t)r(t)=sm​(t)+n(t)

其中，r(t)r(t)r(t) 是接收到的信号，sm(t)s_m(t)sm​(t) 是经过调制的信号，n(t)n(t)n(t) 是高斯白噪声。这里使用的是所谓的白噪声，是其功率谱密度Sn(f)=N02S_n(f)=\frac{N_0}{2}Sn​(f)=2N0​​(W/Hz)，均值为 0。

尽管 AWGN 信道非常简单，但是其作为一个最基本的信道模型，能够很好的描述噪声的影响，也是其他更加复杂的信道模型的基础。

前面提到了通过标准正交基，可以用s\boldsymbol{s}s 表述每一个信号sm(t)s_m(t)sm​(t)，而噪声也可以用标准正交基来表示，得到一个各分量 I.I.D、均值为 0、方差为N0/2N_0/2N0​/2 的矢量形式n\boldsymbol{n}n，这样整个信号可以表述为r=s+n\boldsymbol{r}=\boldsymbol{s}+\boldsymbol{n}r=s+n。

# 最佳检测

一个接收机在接收到信号之后，对信号的处理一般是两个步骤，也就是解调 demodulate 和判决 detect。

对于输入的信号r(t)r(t)r(t)，接收机将其 demodulate，得到一个NNN 维矢量r=[r1,r2,...,rN]\boldsymbol{r}=[r_1,r_2,...,r_N]r=[r1​,r2​,...,rN​]，这个过程也就是一个向量化的。

而针对这个向量化的信号，接收机使用一个判决函数对其进行分析，判断这个矢量对应哪一个消息m^\hat{m}m^。这个判决函数为一个RN\mathbb{R}^NRN 到{1,2,...,M}\{1,2,...,M\}{1,2,...,M} 的映射，可以表示为

g(r)=m,if r∈Dmg(\boldsymbol{r})=m,if\ \boldsymbol{r}\in\mathcal{D}_mg(r)=m,if r∈Dm​

这里的Dm\mathcal{D}_mDm​ 被称为判决域（Decision Region）。

而一个最佳接收机的目的也就是使得这个判断过程的错误概率最低。

很明显，正确判断的概率是

Pc=∑m=1Pr{sm sent}∫Dmf(r∣sm)dr=∑m=1M∫Dmf(r)Pr{sm sent∣r received}P_c=\sum_{m=1}Pr\{\boldsymbol{s}_m\ sent\}\int_{\mathcal{D}_{m}}f(\boldsymbol{r}|\boldsymbol{s}_m){\rm d}\boldsymbol{r} =\sum_{m=1}^{M}\int_{\mathcal{D}_m}f(\boldsymbol{r})Pr\{\boldsymbol{s}_m\ sent|\boldsymbol{r}\ received\}Pc​=m=1∑​Pr{sm​ sent}∫Dm​​f(r∣sm​)dr=m=1∑M​∫Dm​​f(r)Pr{sm​ sent∣r received}

这里的fff 是概率密度函数，式中第一步为先验概率，第二步中为后验概率。很明显，理想的判决方式就是使得正确判断概率最大，即

gMAP(r)=arg⁡max⁡1≤m≤MPr{sm sent∣r received}g_{MAP}(\boldsymbol{r})=\arg\max_{1\le m\le M}Pr\{\boldsymbol{s}_m\ sent|\boldsymbol{r}\ received\}gMAP​(r)=arg1≤m≤Mmax​Pr{sm​ sent∣r received}

这个也就是所谓的最大后验概率（maximum a posteriori probability, MAP）准则，利用 Bayes 公式，能够得到

gMAP(r)=arg⁡max⁡1≤m≤MPr{sm sent∣r received}=arg⁡max⁡1≤m≤MPr{sm}Pr{r∣sm}Pr{r}g_{MAP}(\boldsymbol{r})=\arg\max_{1\le m\le M}Pr\{\boldsymbol{s}_m\ sent|\boldsymbol{r}\ received\}=\arg\max_{1\le m\le M}\frac{Pr\{\boldsymbol{s}_m\}Pr\{\boldsymbol{r}|\boldsymbol{s}_m\}}{Pr\{\boldsymbol{r}\}}gMAP​(r)=arg1≤m≤Mmax​Pr{sm​ sent∣r received}=arg1≤m≤Mmax​Pr{r}Pr{sm​}Pr{r∣sm​}​

如果发送的符号是等概率的，即Pr{sm}=1MPr\{\boldsymbol{s}_m\}=\frac{1}{M}Pr{sm​}=M1​，上面的式子变成

gML(r)=arg⁡max⁡1≤m≤MPr{r∣sm}g_{ML}(\boldsymbol{r})=\arg\max_{1\le m\le M}Pr\{\boldsymbol{r}|\boldsymbol{s}_m\}gML​(r)=arg1≤m≤Mmax​Pr{r∣sm​}

这个被称为最大似然（maximum likelihood, ML）准则。

ML 准则要求发送的符号是等概率的，如果条件不满足，ML 接收机并不是最佳的。但是实际中，MAP 需要准确知道发送消息的概率，这是比较困难的，因此 ML 是工程上一个比较好的选择。

# 差错概率

对于接收机，接收的符号错误率有时候比正确率更加重要。一个最基本的衡量标准就是符号错误率（Symbol Error Rate, SER）：

Pe=∑m=1MPmPr{g(r)≠m∣sm sent}=∑m=1MPm∑m′≠m∫Dm′f(r∣sm)drP_e=\sum_{m=1}^{M}P_mPr\{g(\boldsymbol{r})\neq m|\boldsymbol{s}_m \ sent\}\\[2ex] =\sum_{m=1}^{M}P_m\sum_{m'\neq m}\int_{\mathcal{D}_m'}f(\boldsymbol{r}|\boldsymbol{s}_m){\rm d}\boldsymbol{r}Pe​=m=1∑M​Pm​Pr{g(r)=m∣sm​ sent}=m=1∑M​Pm​m′=m∑​∫Dm′​​f(r∣sm​)dr

Pr{bi=l∣r}=∑sm:bi=lPr{sm∣r}Pr\{b_i=l|\boldsymbol{r}\}=\sum_{\boldsymbol{s}_m:b_i=l}Pr\{\boldsymbol{s}_m|\boldsymbol{r}\}Pr{bi​=l∣r}=sm​:bi​=l∑​Pr{sm​∣r}

这里的概率包括了该符号正确的概率和符号错了但是对应的 bit 正确的概率，这种情况下的 MAP 判决为

gMAPi(r)=arg⁡max⁡l∈{0,1}∑sm:bi=lPr{sm∣r}g_{MAPi}(\boldsymbol{r})=\arg\max_{l\in\{0,1\}}\sum_{s_m:b_i=l}Pr\{\boldsymbol{s}_m|\boldsymbol{r}\}gMAPi​(r)=argl∈{0,1}max​sm​:bi​=l∑​Pr{sm​∣r}

# 充分统计量

我们考虑发送的消息为sm\boldsymbol{s}_msm​，接收机得到r=(r1,r2)\boldsymbol{r}=(r_1,r_2)r=(r1​,r2​)，如果

Pr[r∣sm]=Pr[r1,r2∣sm]=Pr[r1∣sm]Pr[r2∣r2]Pr[\boldsymbol{r}|\boldsymbol{s}_m]=Pr[r_1,r_2|\boldsymbol{s}_m]=Pr[r_1|\boldsymbol{s}_m]Pr[r_2|r_2]Pr[r∣sm​]=Pr[r1​,r2​∣sm​]=Pr[r1​∣sm​]Pr[r2​∣r2​]

这样形成了一个 Markov 链sm→r1→r2\boldsymbol{s}_m\rightarrow r_1\rightarrow r_2sm​→r1​→r2​。

在这种情况下，我们不需要使用r2r_2r2​ 就可以进行理想判决：

gopt(r)=arg⁡max⁡1≤m≤MPr{sm∣r}=arg⁡max⁡1≤m≤MPr{sm}Pr{r∣sm}=arg⁡max⁡1≤m≤MPr{sm}Pr{r1∣sm}Pr{r2∣r1}=arg⁡max⁡1≤m≤MPr{sm}Pr{r1∣sm}g_{opt}(\boldsymbol{r})=\arg\max_{1\le m\le M}Pr\{\boldsymbol{s}_m|\boldsymbol{r}\}=\arg\max_{1\le m\le M}Pr\{\boldsymbol{s}_m\}Pr\{\boldsymbol{r}|\boldsymbol{s}_m\}\\[2ex] =\arg\max_{1\le m\le M}Pr\{\boldsymbol{s}_m\}Pr\{r_1|\boldsymbol{s}_m\}Pr\{r_2|r_1\}\\[2ex] =\arg\max_{1\le m\le M}Pr\{\boldsymbol{s}_m\}Pr\{r_1|\boldsymbol{s}_m\}gopt​(r)=arg1≤m≤Mmax​Pr{sm​∣r}=arg1≤m≤Mmax​Pr{sm​}Pr{r∣sm​}=arg1≤m≤Mmax​Pr{sm​}Pr{r1​∣sm​}Pr{r2​∣r1​}=arg1≤m≤Mmax​Pr{sm​}Pr{r1​∣sm​}

这样子，我们称r1r_1r1​ 为充分统计量（sufficient statistic），r2r_2r2​ 为无关统计量（irrelevant statistic）。
